# Ardha Guides

This directory contains comprehensive guides for using and developing with Ardha. Each guide focuses on a specific aspect of the system, providing detailed explanations, examples, and best practices.

## Available Guides

### Core System Guides

- **[Authentication System](authentication.md)** - User management, JWT tokens, OAuth integration
- **[AI Integration](ai-integration.md)** - AI workflows, model selection, prompt engineering
- **[Database System](database-system.md)** - Notion-style databases, queries, relationships
- **[Git Integration](git-integration.md)** - Version control, commit tracking, task synchronization

### Feature Guides

- **[Notifications](notifications.md)** - Real-time notifications, email delivery, preferences
- **[Background Jobs](background-jobs.md)** - Celery tasks, job processing, monitoring

### Development Guides

- **[API Development](api-development.md)** - Building API endpoints, testing, documentation
- **[Frontend Development](frontend-development.md)** - React components, state management, UI patterns
- **[Workflow Development](workflow-development.md)** - Creating custom AI workflows, LangGraph integration

## Guide Structure

Each guide follows a consistent structure:

1. **Overview** - High-level introduction to the feature
2. **Architecture** - Technical design and implementation details
3. **Usage** - How to use the feature with examples
4. **Configuration** - Setup and customization options
5. **Best Practices** - Recommended patterns and approaches
6. **Troubleshooting** - Common issues and solutions
7. **API Reference** - Relevant API endpoints and schemas

## Getting Started

If you're new to Ardha, we recommend reading the guides in this order:

1. [Authentication System](authentication.md) - Understand user management
2. [AI Integration](ai-integration.md) - Learn about AI capabilities
3. [Database System](database-system.md) - Master data management
4. [Git Integration](git-integration.md) - Integrate with version control
